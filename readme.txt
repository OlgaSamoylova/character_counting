Для запуска приложения необходимо загрузить код из репозитория и запустить его
Для проверки работы необходимо зайти в postman,
    создать POST-request http://localhost:8080,
    указать в BODY строку для получения результата в формате JSON
    пример:
        {
            "string": "Ваша строка"
        }
    отправить запрос,
    в ответе будет получен JSON с символами и их количеством в строке без учета регистра
    пример:
    {
        "а": 3,
        " ": 1,
        "р": 1,
        "с": 1,
        "в": 1,
        "т": 1,
        "ш": 1,
        "к": 1,
        "о": 1
    }
